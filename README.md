# svim 
Recreation of the vi text editor in svelte 

## Supported Features
- Normal mode 
- Insert mode 
- Visual mode 

## Planned Features
- Marks
- More customization options
- Visual block mode
- Options mode (c/d)
- Macros 
- Find and replace
- Undo/Redo
- yank/paste

## Notes: 
The regex used to determine word and WORD (w/e/b/B) is not 100% accurate to the original vi editor
