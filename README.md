# svim 
Recreation of the vi text editor in svelte 

## Supported Features
- Normal mode 
- Insert mode 
- Visual mode 
- yank/paste
- Macros 
- Find/next
- Some Options mode (c/d/y)

## Planned Features
- Replace
- Multi Insert
- Find and replace
- Undo/Redo
- Marks
- More customization options
- Visual block mode

## Notes: 
This is only a RECREATION of the vi text editor and not a 1 to 1 exact copy 
don't expect your favourite mystical keybindings to work as it would on vi.
